Usage

To use the application, first login into the system. Only verified inspectors of our internal system can login. If you wish to become an inspector and gain access, please navigate to the help page and click on the email link to send us a mail.

Once you have logged in, you will be greeted with our home page. This page consists of all the apropriate navigation to certain functionality. For example, navigate to the Cars page. Once on the cars page, you will have 4 main functionality to use. You may read, delete, update, and insert data into our production database. To read data, simply click read. The data will appear on screen. To delete a row from the database, simply type in the correct car number you would like to delete (make sure that you read the database first to identify which is the correct number) and then click the delete button. To update an entry, type in the details of the car you would like to update. Note that you cannot change the car number of a vehicle. To insert data into the database, simply read from the database, if the last entry has a car number of 13, then insert 14 into the text fields and continue adding an entry.

All actions are deemed successful if they are added, deleted, updated, and read from the database. This can be seen whereby the application will provide you with confirmation if the action is done.

Note that all pages will provide the user with the ability to go back to the home page of the application. Remember that if you do not know what to do, navigate to the home button and wait for our internal technicians to assist you.
